# MVP session checklist

Use this for issue **010-mvp-readiness** and **MVP success** in `CONTEXT.md`.

## Setup

- [ ] Exactly **two** displays connected; arrangement matches System Settings (left = 1, right = 2)
- [ ] `swift build` succeeds; run app: `swift run EyeWindow`
- [ ] **Camera** and **Accessibility** granted in System Settings

## 30+ minute study session

- [ ] Video or course on one display; Obsidian (or notes app) on the other
- [ ] Start session from menu bar
- [ ] Menu shows dual layout OK (not “Need exactly 2 displays”)
- [ ] Dwell on notes display → focus moves; type without Command+Tab for typical note-taking
- [ ] Brief glance at video while typing notes does **not** steal focus (focus lock)
- [ ] Menu indicator shows `1` or `2` matching the display you are working on
- [ ] Use **Gaze pause** (Control+Option+`) at most occasionally
- [ ] Command+Tab corrections are rare

## Out of scope (confirm absent)

- [ ] No calibration wizard
- [ ] No third-display switching
- [ ] No cloud / network upload
- [ ] No login item auto-start
